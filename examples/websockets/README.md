# Description

A base application developed with the [Serinus](https://serinus.app) framework.

# Getting Started

To run this application, make sure you have [Dart SDK](https://dart.dev/get-dart) installed (version 3.9.0 or higher is required).

## Install dependencies

```bash
dart pub get
```

## Run the application

```bash
serinus run
```

## Deployment

To deploy the application, you can use the following command:

```bash
serinus deploy
```

It will generate a docker image that you can run on any server with Docker installed.
